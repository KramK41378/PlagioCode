# PlagioCode
PlagioCode - приложение для проверки на плагиат. Оно представляет из себя интерфейс с 4 алгоритмами для сверки кода.

# Актуальность
  При изучении программирования у многих учеников может возникнуть желание списывать программы у своих друзей, одноклассников или у генеративных моделей. При списывании качество понимания сильно падает и становится невозможным понять некоторые темы программирования. Учителю необходимо проверять большое количество работ учеников и дополнительная проверка на списывание отнимает много времени и сил. 
  Созданное приложение поможет преподавателям снизить нагрузку, освободит от большого количества ручной проверки кода и высвободит время под объяснение материала.

# Цель
Создание приложения, позволяющего оценивать плагиат учеников, должно уметь сравнивать программный код различных учеников одного класса и определять процент совпадения двух программ.
Очевидно, ученики во многом полагаются на генеративные сети, поэтому основной задачей стало не определение процента совпадения кодов программ нескольких учеников, но сопоставление программного кода, написанного нейросетями с кодом, предоставленным учеником.
Одной основных проблем при написании приложения стало то, что уже разработанные алгоритмы крайне сложны в реализации, а процесс вычисления крайне трудоемок и сложен.

# Результат и описание
  Результатом нашей работы можно считать приложение, способное посчитать и вывести процент соответствия двух программных кодов, написанных на языке Python. Приложение можно использовать в сфере образования, для обеспечения честности выполнения заданий. Также продукт может использоваться для решения юридических вопросов, например проверки на несанкционированное использование лицензионного кода в open-source проектах.

# Выявленные проблемы и перспективы дальнейшей работы
  В нашем приложении используется нейросеть от Groq, что не позволяет использовать функцию генерации кода без VPN. Можно найти работающий на территории РФ LLM и использовать его.
  На данный момент приложение работает только с кодом, написанным на языке Python. Приложение можно модифицировать, чтобы на плагиат проверялись программы, написанные на других языках.
